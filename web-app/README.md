# Legal Doc Drafter — Web App

React + Vite single-page application for the Legal Doc Drafter platform.

## Tech Stack

- **React 19** with Vite 8
- **React Three Fiber** — 3D particle network backgrounds, floating wireframe geometrics, shield scene
- **Framer Motion 12** — spring physics page transitions, 3D scroll-reveal animations, micro-interactions
- **GSAP** — scroll-driven storytelling and advanced timeline animations
- **Lenis** — smooth scroll interpolation with inertial momentum
- **Zustand** — lightweight global UI state (toasts, modals, reduced motion)
- **TanStack Query** — API data caching and request management
- **AWS SDK** — S3 uploads, Cognito authentication (custom login flow)
- **Mammoth.js** — DOCX → HTML preview
- **CSS** — glassmorphism, cursor-reactive glow, dark enterprise SaaS theme

## Quick Start

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
```

Output goes to `dist/`. Upload `dist/` to your S3 bucket for static hosting.

**Tip:** Set `Cache-Control: no-cache` on `index.html` to prevent browser caching issues:
```bash
aws s3 sync dist/ s3://YOUR_BUCKET --delete
aws s3 cp dist/index.html s3://YOUR_BUCKET/index.html --cache-control "no-cache, no-store, must-revalidate" --content-type "text/html"
```

## Project Structure

```text
src/
  auth/           # Cognito auth context, sign-in/sign-up helpers
  components/     # Navbar, AnimatedButton, AnimatedInput, GlassCard,
                  # Modal, Toast, SkeletonLoader, ProcessingPipeline,
                  # PageTransition
  lib/            # motion.js (spring presets, animation variants),
                  # useSmoothScroll.js (Lenis integration)
  pages/          # LandingPage, LoginPage, DashboardPage (+ CSS)
  store/          # Zustand UI store (toasts, modals, reduced motion)
  theme/          # tokens.css (design tokens, CSS variables, keyframes)
  three/          # HeroCanvas, LegalNetworkScene (particles),
                  # FloatingGeometrics (wireframes), ShieldScene
  App.jsx         # Lazy-loaded routes, QueryClient, Lenis, Toast container
  config.js       # Generated by scripts/sync_config.py
  index.css       # Global design tokens and utilities
  main.jsx        # Entry point with reduced-motion detection
```

## Configuration

Client config is generated via the interactive setup script:

```bash
python scripts/setup.py
```

This produces `src/config.js` with AWS Cognito, S3, and API configuration.

## Design Features

- **Fixed 3D particle network** — React Three Fiber canvas behind all page sections
- **Floating wireframe geometrics** — depth-based parallax polyhedra with mouse reactivity
- **3D scroll-reveal cards** — perspective `rotateX`/`rotateY` flip-up with staggered delays
- **Glassmorphism panels** — `backdrop-filter: blur(24px)` with depth layering
- **Spring physics** — all interactions use tuned spring configs (snappy, smooth, bouncy, gentle)
- **Cursor-reactive glow** — radial gradient follows mouse across dashboard
- **Toast notification system** — spring-animated stacking toasts via Zustand
- **Modal system** — scale + blur + focus trap with Escape close
- **Skeleton loaders** — shimmer animation for loading states
- **Smooth scroll** — Lenis inertial scrolling with native IntersectionObserver support
- **Processing pipeline** — animated 5-step flow (Upload → Extract → Structure → Draft → PDF)
- **3D shield** — rotating wireframe icosahedron on login with mouse parallax
- **Reduced motion support** — respects `prefers-reduced-motion` system setting
- **Professional SVG icons** — zero emojis, all Lucide-style line icons

## Auth Flow

Uses AWS Cognito custom login (not Hosted UI). The app client must have:

- `ALLOW_USER_PASSWORD_AUTH` enabled
- `Generate client secret` **disabled**
